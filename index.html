Инструкция по использованию конструктора

  1. Установите нужный размер фона (1920×768 для групп vk)
  2. Установите размер фона с помощью полей Ширина и Высота, затем нажмите Размер.
  3. Загрузите фоновое изображение (PNG или JPG) через кнопку Загрузить Фон.
  4. Загрузите overlay через кнопку Загрузить Overlay (PNG или JPG).
  5. overlay это (логотип, текст, в формате PNG или JPG )
  6. Перетаскивайте overlay мышкой по холсту.
  7. Настройте размер overlay с помощью регулятора. рядом с кнопкой Overlay.
  8. Выберите формат (PNG или JPG) и нажмите Скачать.
